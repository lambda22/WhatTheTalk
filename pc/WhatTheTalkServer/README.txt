How to compile?
1. Change directory to pc/WhatTheTalkServer
2. Run "javac -sourcepath src -cp jar\* -d bin src\com\wtt\server\Main.java"

How to use?
1. Change directory to pc/WhatTheTalkServer
2. Run "java -cp bin;jar\* com.wtt.server.Main [PORT]"
3. Close this program by entering "exit"
** Default PORT is 5566.
